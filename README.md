## CricketConnect Pro

A comprehensive cricket team management platform for organizing teams, booking grounds, scheduling matches, and managing tournaments.

## Features

- **Team Management**: Create and manage cricket teams with player rosters
- **Ground Booking**: Reserve cricket grounds with real-time availability
- **Player Network**: Connect with players and build your cricket community
- **Match Scheduling**: Organize matches and tournaments
- **Real-time Chat**: Communicate with team members
- **Profile Management**: Customize your cricket profile and settings

## Getting Started

To run the development server:
```
npm install
npm run dev
```

## Deployment

### Deploy to Vercel
```
npm run deploy:vercel
```

### Deploy to Cloudflare Workers
```
npm run deploy
```

For detailed deployment instructions, see `DEPLOY_TO_VERCEL.md` or the Cloudflare Workers documentation.
